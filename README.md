# KIRA 个人项目仓

## 介绍

仓库分为两个部分：独立项目与功能组件。其中独立项目作为单独文件夹存在，功能组件统一保存在 Components 文件夹中。
## 项目介绍

玄音音乐：基于 HarmonyOS 开发的原生音乐播放器应用，具备工业级细节处理，UI 风格对标主流音乐应用。  
待办任务：基于 ArkTS 开发的智能任务管理应用，实现基于时间轴的动态优先级评估与本地数据全量持久化。
